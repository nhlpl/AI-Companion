We'll design a living fractal avatar that evolves with the conversation. The avatar will be a separate Macroquad process, drawing a rotating fractal tree whose complexity, colors, and motion respond to the AI's state.

---

## 🧬 Fractal Design

| Parameter | Mapped From | Visual Effect |
|:---|:---|:---|
| **Recursion depth** | `session_message_count` | More branches = longer conversation |
| **Branch angle** | `trust` | Higher trust = wider, more open branches |
| **Hue** | `emotion` | Each emotion has a base color |
| **Saturation** | `trust` | Higher trust = more vibrant |
| **Rotation speed** | `mode` + `speaking` | Different archetypes spin at different speeds; speaking accelerates |
| **Pulse** | `speaking` | Branches "breathe" when AI is talking |

## 📁 Avatar Process Code (`avatar/src/main.rs`)

```rust
use macroquad::prelude::*;
use serde::{Deserialize, Serialize};
use std::f32::consts::PI;
use tokio::net::TcpListener;
use tokio::sync::watch;
use std::sync::Arc;

// ---------- State from AI ----------
#[derive(Debug, Deserialize, Serialize, Clone)]
struct AvatarState {
    emotion: String,
    trust: f32,
    mode: String,
    speaking: bool,
    message_count: u32, // total messages exchanged
}

impl Default for AvatarState {
    fn default() -> Self {
        Self {
            emotion: String::new(),
            trust: 0.5,
            mode: "Companion".to_string(),
            speaking: false,
            message_count: 0,
        }
    }
}

// ---------- Fractal Tree ----------
fn draw_branch(
    x: f32,
    y: f32,
    length: f32,
    angle: f32,
    depth: u32,
    max_depth: u32,
    hue: f32,
    saturation: f32,
    time: f32,
) {
    if depth == 0 {
        return;
    }

    let end_x = x + length * angle.cos();
    let end_y = y + length * angle.sin();

    // Color based on depth (darker at roots, brighter at tips) and time pulse
    let lightness = 0.4 + 0.3 * (depth as f32 / max_depth as f32);
    let pulse = if depth % 2 == 0 {
        0.1 * (time * 2.0).sin()
    } else {
        0.0
    };
    let color = Color::from_hsl(hue, saturation, lightness + pulse);

    // Line thickness decreases with depth
    let thickness = 2.0 * (depth as f32 / max_depth as f32) + 1.0;
    draw_line(x, y, end_x, end_y, thickness, color);

    // Recursive branches
    let new_length = length * 0.7;
    let angle_offset = PI / 6.0; // 30 degrees

    draw_branch(
        end_x,
        end_y,
        new_length,
        angle - angle_offset,
        depth - 1,
        max_depth,
        hue,
        saturation,
        time,
    );
    draw_branch(
        end_x,
        end_y,
        new_length,
        angle + angle_offset,
        depth - 1,
        max_depth,
        hue,
        saturation,
        time,
    );
}

// ---------- Main ----------
#[macroquad::main("AI Companion Avatar")]
async fn main() {
    // TCP server for receiving state
    let listener = TcpListener::bind("127.0.0.1:9001").await.unwrap();
    println!("Avatar listening on 127.0.0.1:9001");

    // Watch channel for state updates
    let (tx, mut rx) = watch::channel(AvatarState::default());
    let tx = Arc::new(tx);

    // Spawn connection handler
    tokio::spawn(async move {
        loop {
            if let Ok((mut stream, _)) = listener.accept().await {
                let tx = tx.clone();
                tokio::spawn(async move {
                    let mut buf = vec![0u8; 1024];
                    loop {
                        match stream.try_read(&mut buf) {
                            Ok(n) if n > 0 => {
                                if let Ok(state) = serde_json::from_slice::<AvatarState>(&buf[..n]) {
                                    let _ = tx.send(state);
                                }
                            }
                            _ => break,
                        }
                    }
                });
            }
        }
    });

    let mut time = 0.0;
    let mut rotation = 0.0;

    loop {
        time += 0.016; // approx 60fps

        // Get latest state (non-blocking)
        let state = rx.borrow().clone();

        // Compute visual parameters
        let hue = match state.emotion.as_str() {
            "frustration" => 0.0,   // red
            "sadness" => 0.6,        // blue
            "excitement" => 0.15,    // yellow-orange
            "anxiety" => 0.8,        // purple
            "confusion" => 0.0,      // gray (will be desaturated)
            _ => 0.3,                // greenish default
        };

        let saturation = if state.emotion == "confusion" {
            0.1
        } else {
            0.5 + state.trust * 0.5
        };

        // Complexity: recursion depth grows with message count
        let max_depth = 2 + (state.message_count / 15).min(6) as u32;

        // Rotation speed
        let base_speed = match state.mode.as_str() {
            "Mentor" => 0.2,
            "Trickster" => 0.8,
            _ => 0.4,
        };
        let speed = if state.speaking { base_speed * 2.0 } else { base_speed };
        rotation += speed * 0.016;

        clear_background(BLACK);

        let center_x = screen_width() / 2.0;
        let center_y = screen_height() / 2.0;
        let trunk_length = 120.0;

        // Draw fractal tree
        draw_branch(
            center_x,
            center_y + 50.0, // start slightly below center
            trunk_length,
            -PI / 2.0 + rotation * 0.1, // trunk points up, with slow rotation
            max_depth,
            max_depth,
            hue,
            saturation,
            time,
        );

        // Optional: draw a second mirrored tree or a glow effect
        draw_circle(center_x, center_y, 5.0, WHITE);

        next_frame().await;
    }
}
```

## 🔌 Integration with Dioxus App

Add `message_count` to the `AvatarState` and update it on each message.

### `src/avatar_client.rs` (updated)

```rust
#[derive(Debug, Serialize, Deserialize, Clone)]
pub struct AvatarState {
    pub emotion: String,
    pub trust: f32,
    pub mode: String,
    pub speaking: bool,
    pub message_count: u32,
}
```

In `AppState`, increment a counter on each user message and send updates:

```rust
let mut message_count = 0;
// In send_message:
message_count += 1;
let avatar_state = AvatarState {
    emotion: emotion.clone(),
    trust: self.settings_engine.read().trust_level,
    mode: format!("{:?}", *self.current_archetype.read()),
    speaking: true,
    message_count,
};
avatar_client.send_state(&avatar_state);
```

After response, send again with `speaking: false`.
